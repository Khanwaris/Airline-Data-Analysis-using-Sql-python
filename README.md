 Airline Data Analysis using SQL & Python

Project Overview
This project focuses on analyzing airline operational data to understand flight performance in terms of revenue generation and seat occupancy. The goal is to identify inefficiencies, detect high-demand routes, and provide data-driven recommendations to improve overall profitability.

---

 Business Problem
The airline is facing challenges in maintaining consistent profitability due to:
- Low occupancy in certain flights  
- Inefficient route planning  
- Fluctuating customer demand  

At the same time, some routes show very high demand, indicating an imbalance in capacity and pricing strategy.



 Objectives
- Increase flight occupancy rate  
- Optimize revenue and pricing strategies  
- Identify high-demand and low-demand routes  
- Improve overall operational efficiency  



 Tools & Technologies
- SQL (SQLite)  
- Python (Pandas, Matplotlib)  



Key Analysis
- Revenue analysis (top & bottom flights)  
- Occupancy analysis (high vs low utilization)  
- Route analysis (high-demand & low-demand routes)  



Key Insights
- Many flights operate at low occupancy  
- Some flights run at full capacity (high demand)  
- Revenue is concentrated in key routes  
- DME ↔ KHV routes show highest revenue  
- Some routes show very low demand  



Business Recommendations
- Optimize low-occupancy flights  
- Increase capacity on high-demand routes  
- Apply dynamic pricing strategies  
- Focus on high-revenue routes  
- Re-evaluate low-performing routes  



Conclusion
This project shows how data analysis can help improve airline efficiency and profitability using SQL and Python.
